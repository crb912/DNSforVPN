用户已日常运行: config.toml port=1553 (免 sudo), dig 实测覆盖/上游均正常。

Architecture (5 layers + router):
- internal/dns: wire-format parse/build — 17 tests (含 CNAME rdata 终止符回归)
- internal/router: GFWList 路由, Route() → RouteResult — 16 tests
- internal/cache: Cache(Get/Set/Del/List) + CustomStore(bucket "custom",
  CustomEntry{Domain, IPs}) + NegativeCache, BoltDB 单文件双 bucket — 20 tests
- internal/upstream: Resolver + doh/udp Client + Manager
  (ResolveAll 聚合 + reduce 按 Name|Type|RData 去重) — 4 tests
- internal/query: Service.Handle 管线: 1b custom 覆盖(最高优先) → route →
  inflight dedup → negCache → cache → upstream; New(c, nc, r, u, cs)
- internal/transport: UDPServer

Root: main.go (Wails, embed frontend/dist + AssetServer),
app.go (13 Bind 方法, withStore: running 用活句柄/stopped 临时开库),
cmd/dnsforvpn/main.go (CLI headless)

Frontend (Svelte 4, 5 tab): Config/Latency/Stats/Cache/CustomDNS 面板,
wailsjs 绑定由 wails dev 自动生成。
